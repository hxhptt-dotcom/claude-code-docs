# Claude Code 官方文档 - 大白话版本

> 📘 **Claude Code 中文大白话文档** - 用最通俗易懂的语言解释 Claude Code 的所有功能

## 📖 关于本项目

这是 [Claude Code 官方文档](https://code.claude.com/docs/zh-CN)的非官方中文大白话版本。官方文档虽然详细,但术语较多,初学者难以理解。本项目用通俗的语言解释每个功能,帮助你快速上手!

### ✨ 核心特点

- ✅ **通俗易懂** - 用大白话解释每个功能,无需专业背景
- ✅ **示例丰富** - 每个功能都有实际使用示例
- ✅ **最佳实践** - 总结了使用技巧和常见陷阱
- ✅ **本地访问** - 双击即可打开,无需网络
- ✅ **搜索功能** - 快速查找你需要的命令和配置
- ✅ **收藏功能** - 标记常用功能,随时查看
- ✅ **代码复制** - 一键复制代码示例
- ✅ **深色模式** - 支持亮色/暗色主题切换

## 🎯 功能使用指南

### 🔍 搜索功能

- **快捷键**: `Ctrl+K` (Windows/Linux) 或 `Cmd+K` (Mac)
- **使用方法**: 在顶部搜索框输入关键词即可
- **搜索范围**: 命令、配置、功能说明

### ⭐ 收藏功能

- **添加收藏**: 点击任何功能块底部的 "⭐ 收藏此功能" 按钮
- **查看收藏**: 在左侧边栏底部的 "⭐ 我的收藏" 面板中查看
- **删除收藏**: 在收藏面板中点击 "✕" 按钮
- **数据存储**: 收藏数据保存在浏览器 LocalStorage 中

### 📋 代码复制

- **使用方法**: 点击任何代码块右上角的 "📋 复制" 按钮
- **反馈提示**: 复制成功后会显示 "✓ 已复制!"
- **兼容性**: 支持所有现代浏览器

### 🌙 主题切换

- **切换方法**: 点击顶部导航栏的 "🌙" 按钮
- **自动模式**: 系统会根据你的设置自动选择亮色/暗色主题
- **记忆功能**: 你的选择会被保存,下次打开时自动应用

## 🛠️ 自定义和扩展

### 添加新内容

1. 用文本编辑器打开 `index.html`
2. 找到对应章节的 `<section>` 标签
3. 按照现有格式添加新内容

### 修改样式

所有样式都在 `<style>` 标签中,可以修改 CSS 变量来调整配色:

```css
:root {
    --primary-color: #CC785C;      /* 主色调 */
    --bg-primary: #FFFFFF;         /* 主背景 */
    --text-primary: #2C3E50;       /* 主文字色 */
}
```

### 添加新的 Slash 命令

在 Slash 命令章节中添加新的 `<article>` 标签:

```html
<article id="your-command" class="example" style="border-left-color: var(--primary-color); margin-top: 30px;">
    <h3>/your-command - 命令描述</h3>
    <p class="desc">大白话解释...</p>
    <!-- 更多内容 -->
</article>
```

## 🔧 故障排除

### 搜索功能不工作

- 确保浏览器支持 JavaScript
- 检查浏览器控制台是否有错误
- 尝试刷新页面 (Ctrl+F5)

### 收藏功能失效

- 确保浏览器支持 LocalStorage
- 检查是否在隐私/无痕模式下
- 清除浏览器缓存后重试

### 代码复制失败

- 更新浏览器到最新版本
- 检查浏览器权限设置
- 尝试手动复制 (Ctrl+C)

### 样式显示异常

- 确保完整下载了 HTML 文件
- 检查网络连接(如果有外部资源)
- 尝试其他浏览器
